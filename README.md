# 🩺 1분 CPX (CPX Training Web App) v2.2

**"CPX를 땡시처럼!"**
🔔 해부학 땡시 종이 울리고 빠른 시간에 답을 떠올려 적어내듯, **주소(C.C)를 보자마자 무엇을 물어봐야 할지 반사적으로 떠오르도록** 훈련하는 모바일 최적화 웹앱입니다.

우선, 해당 앱은 실기시험 자체를 완벽히 대비하는 앱이 아닙니다.
반드시 동기들과 만나서 CPX를 해보는 연습이 메인이 되어야 한다는 점을 강조드리고 싶습니다.
한권으로 끝내기, 유니온 등 실기 준비 연습서의 암기법을 효과적으로 암기하고 떠올리기 위한 보조 수단으로 활용하시면 큰 도움이 될 것이라 생각합니다.

제가 시험을 치르는 26년도 3월 시험 이후론 지시문 상황 숙지 시간이 없어진다고 하여, 이런 훈련이 더 중요해진다는 생각이 듭니다. 이 앱이 조금이라도 도움이 되셨으면 좋겠습니다!

## ✨ 주요 기능 설명 및 활용법

### 1. 📖 학습 모드
목록에서 선택된 카드들을 빠르게 학습하기 위한 모드입니다.
> 💡 **제작자 Tip:** 실기 준비를 본격적으로 시작하기 전에 주소별로 A만 떠올리는 훈련을 하는 데 사용했습니다. 우선 순서대로 익혀보고 셔플 버튼을 눌러 랜덤하게도 학습해 보세요.

### 2. 📝 퀴즈 모드
원하는 계통과 문제 수를 설정하여 나만의 미니 모의고사를 생성하며, 결과 화면에서 틀린(X) 문제만 따로 모아 즉시 재시험을 볼 수 있는 모드입니다.
> 💡 **제작자 Tip 1:** 1주차(1회독) 연습을 하면서 다음 날 진행될 주소들에 대하여 A에 더불어 C, F, PEx를 떠올리는 훈련을 하는 데 사용했습니다.
> 💡 **제작자 Tip 2:** 2주차(2회독)를 할 땐 A, C, F, PEx에 더불어 Edu의 내용까지 떠올리는 훈련을 하는 데 사용했습니다. 

### 3. 🔥 실전 모드 (Practical Mode)
전체 카드를 회독하기 위한 목적과 실전 감각 극대화를 위한 숙련자 전용 모드입니다.
> 💡 **제작자 Tip 1:** 저의 경우 2주차(2회독)가 끝난 뒤 3주차는 회독 모드, 4주차는 파이날 모드로 복습했습니다. 
> 💡 **제작자 Tip 2:** 조원들과 실제로 만나 연습하게 될 증례 스케줄을 랜덤 생성해 주는 스케줄러로도 유용하게 사용했습니다. 다른 조원의 스케줄을 만들고 서로 공유해 주지 않음으로써, 자기가 무슨 주소를 만나게 될지 모르는 상태에서 연습하시는 걸 추천드립니다.

- **📚 회독 모드:** 58개 전체 증례를 누락 없이 가장 빠르게 1회독할 수 있도록 세트를 최소한의 계통별 출제 규칙을 적용하여 자동 분배합니다.
- **🔥 파이날 모드:** 실제 의사국가고시 실기시험 출제 경향(계통별 분배)을 아주 엄격하게 반영하여 9문제씩 8세트의 모의고사를 생성합니다. 주소 수가 적은 계통은 중복 출제될 수 있습니다.
- **📋 일정 공유:** 생성된 스터디 스케줄을 원클릭으로 복사하여 조 카톡방에 바로 공유할 수 있습니다.
- **💾 자동 저장 (이어풀기):** 풀다 만 세트의 진행 상황이 자동 저장되어 다음 접속 시 이어서 학습할 수 있습니다.
  
### 4. 📋 목록 모드
- **🚦 신호등 약점 마킹:** 카드 뒷면의 각 항목(C, A, F, PEx 등) 글자를 터치하면 3단계(🟡 ➔ 🟠 ➔ 🔴)로 색상이 변하여 자신의 취약 파트를 직관적으로 표시할 수 있습니다.
- **약점 스캐너:** 목록 화면에서 각 증례별 정답률과 함께, 내가 자주 틀리는 약점 이니셜(예: `🔴P` `🟡M`)을 한눈에 파악할 수 있습니다.
- **상세 통계:** 각 증례의 정답률 뱃지를 누르면 누적된 O/X 횟수와 상세 퍼센트를 확인할 수 있습니다.

### 5. ⚙️ 환경 설정 및 커스터마이징
상단 톱니바퀴(⚙️) 아이콘을 눌러 나만의 환경을 구축하세요.
- **표기 순서 커스텀:** 카드 뒷면에 표시되는 내용(C, A, F, PEx, Edu, Memo)의 **디폴트 순서를 내 입맛대로 설정하고 전체 카드에 일괄 적용**할 수 있습니다.
- **타이머 설정:** 60초 제한, 무제한, 직접 입력 중 기본 타이머를 세팅할 수 있습니다.

## 💾 데이터 백업 및 연동 (매우 중요)
이 앱은 별도의 서버 없이 사용자의 기기(브라우저 캐시)에 데이터를 저장하는 빠르고 가벼운 앱입니다. **따라서 브라우저 방문 기록이나 캐시를 삭제하면 작성한 나만의 암기법이 모두 날아갑니다.**

- **안전한 백업 (권장):** 설정 ➔ **[💾 파일 다운로드]** 를 통해 주기적으로 데이터를 `.json` 파일 형태로 기기에 저장해 두세요.
- **기기 연동:** 아이패드에서 저장한 파일을 아이폰으로 에어드랍/카톡 전송한 뒤 **[📂 파일 불러오기]**를 하면 모든 기기에서 동일한 데이터를 이어서 학습할 수 있습니다.
