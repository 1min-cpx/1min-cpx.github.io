# 🩺 1분 CPX (CPX Training Web App)

의사 국가고시 실기(CPX) 대비를 위한 **순발력 훈련 및 케이스 스터디 웹앱**입니다.
언제 어디서나 스마트폰으로 1분 안에 주소(C.C)를 보고 연상하는 훈련을 할 수 있습니다.

## 🚀 바로가기 (웹사이트)
👉 **[1분 CPX 실행하기]((https://1min-cpx.github.io/))**
*(위 링크를 클릭하면 앱이 실행됩니다.)*

## ✨ 주요 기능

### 1. 📖 학습 모드 (Study Mode)
- **무작위 증례 생성:** 48개 주제 중 무작위로 케이스를 생성합니다.
- **실전 타이머:** 60초 타이머가 작동하며, 종료 시 화면이 흔들려 시각적으로 알려줍니다. 
                 타이머를 클릭해 원하는 시간 설정이 가능합니다. 
- **카드 뒤집기:** 앞면엔 상황 지침, 뒷면엔 나만의 암기법(Note)을 정리할 수 있습니다.

### 2. 📝 퀴즈 모드 (Mock Exam)
- **범위 선택:** 원하는 계통별 주제(소화기, 순환기 등)만 골라서 퀴즈를 볼 수 있습니다.
- **채점 시스템:** 아는 문제(O)와 모르는 문제(X)를 체크하여 정답률을 확인합니다.
- **오답 노트:** 틀린 문제만 다시 모아서 재시험을 볼 수 있습니다.

### 3. 💾 데이터 관리 (Data Management)
- **내보내기/가져오기:** 서버 없이 **클립보드 복사** 방식으로 데이터를 간편하게 백업합니다.
- **기기 연동:** 아이폰 ↔ 아이패드 ↔ PC 간에 카카오톡 등을 통해 데이터를 쉽게 옮길 수 있습니다.
- **OCR 기능:** 책이나 필기를 사진으로 찍어 텍스트로 자동 입력할 수 있습니다.

## 📱 모바일 최적화
- **반응형 웹:** PC와 모바일 환경에 맞춰 UI가 자동으로 최적화됩니다.
- **앱처럼 사용:** 사파리(iOS)나 크롬(Android)에서 **'홈 화면에 추가'**를 하면 앱처럼 아이콘이 생성됩니다.

## ⚠️ 주의사항
이 앱은 별도의 서버가 없습니다. **인터넷 사용 기록(캐시)을 삭제하면 저장된 데이터가 초기화**될 수 있으니, 중요한 내용은 수시로 **[내보내기]** 하여 백업해 주세요.
